
# ALP Duty Log

An offline-capable Progressive Web App for Indian Railways Assistant Loco Pilots.

## Features
- Offline Duty Logging (Working, Spare, PR, Leave)
- User Profile Management
- Cloud Backup to Google Drive (via System Share)
- Excel (.xlsx) Export
- Searchable History

## Tech Stack
- React
- Vite
- Tailwind CSS
- XLSX.js
